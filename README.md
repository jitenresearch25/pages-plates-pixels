# PPP — Pages • Plates • Pixels

A static GitHub Pages website for cooking, writings, mathematics, and photography.

## Publish on GitHub Pages

1. Upload all files in this folder to a GitHub repository.
2. Go to Settings → Pages.
3. Choose Deploy from branch → main → root.
4. Save and wait for the GitHub Pages link.

## Contact form note

The About page includes a message box that copies a visitor message. Direct form submission can be connected later with Formspree, FormSubmit, Netlify Forms, or a custom domain email.
